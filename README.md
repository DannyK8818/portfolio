# Danny's Portfolio

A clean, modern personal portfolio site built with Tailwind CSS (via CDN for simplicity).

**Live site:** Once you enable GitHub Pages, it'll be at `https://dannyk8818.github.io/portfolio`

## 🚀 Quick Deploy (GitHub Pages)

1. Go to your repo: https://github.com/DannyK8818/portfolio
2. Click **Settings** → **Pages**
3. Under "Build and deployment":
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ (root)**
4. Hit **Save**

Your site should be live in a minute or two at the URL above. Refresh if it says "Your site is ready to be published".

## ✨ What's inside

- Fully responsive clean design
- Smooth scrolling + mobile menu
- Project cards with hover effects
- Working demo contact form (just for show — replace with real form later if you want)
- Subtle keyboard shortcut: Press `/` on the site to jump straight to contact
- Easy to customize — everything is in one `index.html` file

## 🛠 How to customize

Just edit `index.html` directly on GitHub (or clone and edit locally):

- Update bio in the About section
- Replace the fake projects with your real ones (add screenshots, real links, etc.)
- Change skills to match what you're actually using
- Update email and social links in Contact
- Tweak colors (emerald accents are in the CSS)

Want a completely different vibe, more sections, or dark/light toggle? Just tell me and I'll update it.
